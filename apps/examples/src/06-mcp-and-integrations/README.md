# 06 - MCP and Integrations

Learn how to integrate agents with MCP servers and external systems.

## Concepts Covered

- **MCP toolsets** - Connecting to MCP servers
- **Sampling handlers** - Context-aware MCP responses
- **HTTP integration** - Making API requests
- **File operations** - File system management
- **Composite integration** - Combining multiple services

## Running the Example

```bash
pnpm dev --name 06-mcp-and-integrations
```

## Key Takeaways

1. **MCP** enables standardized tool integration
2. **Sampling** allows servers to request client context
3. **HTTP tools** connect to external APIs
4. **File tools** enable file system operations
5. **Composition** combines multiple integrations
