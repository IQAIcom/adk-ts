# 04 - Persistence and Sessions

Learn how to persist data across agent interactions using sessions and artifacts.

## Concepts Covered

- **Database sessions** - Persistent conversation storage
- **Artifact services** - File management and storage
- **Session rewind** - Time-travel through conversation history
- **Event compaction** - Automatic summarization for long conversations
- **Cross-session data** - Sharing data across conversations

## Running the Example

```bash
pnpm dev --name 04-persistence-and-sessions
```

## Key Takeaways

1. **Sessions** persist conversation context and state
2. **Artifacts** enable long-term file storage with versioning
3. **Rewind** allows undoing conversation history
4. **Compaction** reduces token usage in long conversations
