## What You Can Build

### Blockchain Agents

Create agents that can interact with blockchain networks, manage wallets, execute smart contracts, and handle cryptocurrency transactions using Coinbase AgentKit.

### Data-Driven Agents

Connect to databases and data sources to build agents that can query, analyze, and manipulate data across various platforms.

### Communication Agents

Integrate with messaging platforms and communication services to build agents that can send notifications, respond to messages, and coordinate across channels.

### Multi-Platform Agents

Combine multiple integrations to create sophisticated agents that work across different platforms and services simultaneously.

## Contributing

Have you built an integration that others might find useful? We'd love to include it!

**How to contribute:**

- Share your integration guide via [pull request](https://github.com/IQAIcom/adk-ts/pulls)
- Discuss integration ideas in our [community forum](https://github.com/IQAIcom/adk-ts/discussions)
- Report integration issues or suggest improvements

When contributing integration guides, follow the structure of existing guides to maintain consistency.
