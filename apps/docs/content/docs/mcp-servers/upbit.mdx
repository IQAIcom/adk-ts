## Features

- Provides tools for accessing Upbit market data and trading operations
- Supports both public market data (no authentication required) and private trading tools
- Uses Upbit's REST API for reliable and fast cryptocurrency exchange interactions

## Available Tools

The server exposes the following tools that MCP clients can utilize:

### Public Tools (No Authentication Required)

#### `GET_TICKER`
Get the latest ticker information for a specific market.

- Parameters: `market` (string) - Market identifier (e.g., "KRW-BTC")

#### `GET_ORDERBOOK`
Get the current orderbook snapshot for a market.

- Parameters: `market` (string) - Market identifier (e.g., "KRW-BTC")

#### `GET_TRADES`
Get recent trades for a market.

- Parameters: `market` (string) - Market identifier (e.g., "KRW-BTC")

### Private Tools (Requires Authentication)

<Callout type="warning" title="Authentication Required">
Private tools require valid Upbit API credentials and trading to be enabled.
</Callout>

#### `GET_ACCOUNTS`
Get account information and balances.

- Parameters: None

#### `GET_ORDERS`
Get order history with optional filtering.

- Parameters: `market` (string, optional), `state` (string, optional), `page` (number, optional), `limit` (number, optional)

#### `GET_ORDER`
Get specific order details.

- Parameters: `uuid` (string, optional), `identifier` (string, optional)

#### `CREATE_ORDER`
Create a new trading order.

- Parameters: `market` (string), `side` (string), `ord_type` (string), `volume` (string, optional), `price` (string, optional)

#### `CANCEL_ORDER`
Cancel an existing order.

- Parameters: `uuid` (string) - Order UUID to cancel

## Environment Variables

- `UPBIT_SERVER_URL` (optional): Upbit API server URL (defaults to https://api.upbit.com)
- `UPBIT_ACCESS_KEY` (required for private tools): Your Upbit API access key
- `UPBIT_SECRET_KEY` (required for private tools): Your Upbit API secret key
- `UPBIT_ENABLE_TRADING` (required for private tools): Set to "true" to enable trading functionality

## Getting Upbit API Keys

<Callout type="info" title="API Setup Required">
You'll need to create and configure Upbit API keys before using private trading tools.
</Callout>

1. Create an account on [Upbit](https://upbit.com) if you don't already have one
2. Go to the [Upbit Developer Center](https://upbit.com/service_center/open_api_guide)
3. Create a new API key
4. Make sure to set appropriate permissions (read, trade, withdraw as needed)
5. Store your API keys in your environment variables

## Usage Examples

### GET_TICKER

Get the latest ticker for Bitcoin:

```json
{
  "tool_name": "GET_TICKER",
  "arguments": {
    "market": "KRW-BTC"
  }
}
```

### GET_ORDERBOOK

Get the current orderbook for Ethereum:

```json
{
  "tool_name": "GET_ORDERBOOK",
  "arguments": {
    "market": "KRW-ETH"
  }
}
```

### CREATE_ORDER

Create a buy order for Bitcoin:

```json
{
  "tool_name": "CREATE_ORDER",
  "arguments": {
    "market": "KRW-BTC",
    "side": "bid",
    "ord_type": "limit",
    "price": "50000000",
    "volume": "0.001"
  }
}
```

## Error Handling

The tools will return an error message in the `result` field if an error occurs. Common errors include:

- **Missing API keys:** Ensure the required environment variables are set for private tools
- **Invalid market:** Double-check the market identifier format (e.g., "KRW-BTC")
- **Trading disabled:** Set `UPBIT_ENABLE_TRADING=true` to enable trading functionality
- **API rate limits:** Upbit has rate limits that may affect frequent requests
