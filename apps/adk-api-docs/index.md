## Key Features

### Agents
Build powerful AI agents with customizable personalities, tools, and behaviors.

### Models  
Support for multiple LLM providers with unified interface and automatic failover.

### Memory
Persistent memory systems to maintain context across conversations and sessions.

### Tools
Extensible tool system for integrating external APIs, databases, and services.

### Sessions
Session management for tracking conversations, user context, and agent state.

### Events
Event-driven architecture for monitoring, logging, and building reactive systems.
